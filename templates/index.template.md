# Memory Index

> Template. The real `index.md` is generated from frontmatter by
> `scripts/regenerate-index.sh` (and is git-ignored). Do not hand-edit inside the
> AUTOGEN fence — edits there are overwritten on regenerate.

A **roster** of the durable memory files — names/topics + summaries only, no file
paths. Derive locations: a project is `projects/<name>/memory.md`, a domain file is
`domain/<topic>.md`. The active project's `memory.md` (and non-empty `working.md`)
are auto-injected each session; load any other file on demand from this roster.

## Lifecycle
Working memory matures: `projects/<active>/working.md` → `domain/<topic>.md`
(cross-project) or `projects/<active>/memory.md` Decisions Log (project-specific)
via `/promote-memory`.
